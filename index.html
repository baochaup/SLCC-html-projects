Reflection

CSIS-1430 INTERNET & XHTML FUNDAMENTALS

CSIS 1430 course was an interesting and entertaining course. This class served as an introduction to using the Internet & creating Websites. Topics include basic Internet applications & HTML, XHTML, CSS, XML, JavaScript & layout techniques... In this course, I have learned many amazing stuff, such as how to create basic web pages by using HTML tags, how to decorate web pages by using CSS and how to make web pages interactive by using JavaScript. The class itself is really interesting. Professor Stone is really good at teaching as he explains the concepts and confusing things clearly so that students can understand them completely. These fundamental concepts are very important because I will need them to advance to the next course which is Web Programming. Moreover, they are the basic skills that every Web Developers must have. I have realized right away that this class was not only challenging but incredibly fun too. HTML tags are not difficult to understand, but CSS has tons of property and confusing concepts which are really challenging to get all of them. I usually had to spend many hours for CSS while only few hours for HTML. For JavaScript, since I already took a few programming classes, JavaScript was not so hard for me to get it. Overall, my experience for this course is amazing, but I still need a lot of time to study more about advanced techniques in CSS and JavaScript. I would recommend this class to anyone who loves visual animation and web design!
